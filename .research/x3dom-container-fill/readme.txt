x3dom-container-fill
  - A library based on x3dom(http://www.x3dom.org/);
  - Focus on display the result of "Load Planning and Optimization";


Note: To run the demo properly, files should deploy to a http server.
  - If you have python installed, just run:
    - python -m SimpleHTTPServer 8080
  - Then open http://127.0.0.1:8080/ ,  click "unittest.html".


The additional:
  - We include "X3dTooltips.zip" as the handbook, please get it from "docs" folder;
  - JSON is required, please see the folder "JSON-js";
